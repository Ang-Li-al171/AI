Steps to run the runnable jars:

Q1:

java -jar tilepuzal171.jar input1.txt

where "input1.txt" is the input text file name in the current directory.

Input: text file "input1.txt" in the current directory.
Output: text file "tilepuz-al171.txt" in the current directory as well as printed to stdout.

Q2:

java -jar queensal171.jar

and input the integer N from stdin.

Input: integer N from stdin.
Output: board is printed in "queens-al171.csv" in the current directory as well as printed to stdout. Total number of queens that attack each other is printed to stdout.


Source codes are found in q1source-al171.jar and q2source-al171.jar respectively.
